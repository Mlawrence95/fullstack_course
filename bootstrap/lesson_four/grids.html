Grid System

  • The grid system is the mechanism by which Bootstrap is able to organize itself across many different devices and screen sizes.

  • There are twelve column units. These can be combined into units of varying size as long as they add up to 12. ex, 2x 6 column boxes or 3x 4 column boxes.

  • The syntax for specifying a row is col-screensize-number of columns. ex: col-md-6 is a row element for a medium display taking 6 columns

  • You can define how things change during a resize (thresholds are calls breakpoints)
